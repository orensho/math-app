# מתמטיקול - שבילים פלוס

אפליקציה ללימוד מתמטיקה אינטראקטיבית בעברית לכיתות א-ו, מבוססת על תוכנית "שבילים פלוס".

## 🚀 התחלה מהירה

```bash
cd ~/Source/math-app
npm run dev
```

פתח את הדפדפן ב: [http://localhost:3000](http://localhost:3000)

## ✨ תכונות

### 📚 מערכת ניווט מלאה
- **יסודות** - מיומנויות בסיס (לוח כפל ועוד)
- בחירת כיתה (א-ו)
- בחירת ספר לימוד
- רשימת יחידות לימוד
- ניווט בין חלקי היחידה: מושגים, דוגמאות, חידון

### 🎓 תוכן לימודי

**כיתה ו' - שבילים פלוס 16 - כל 6 היחידות (מלאות)**
- יחידה 1: כפל וחילוק שברים - חלק א
- יחידה 2: שוב חישוב - המבנה העשרוני
- יחידה 3: חלק מכמות בשברים
- יחידה 4: שוב חישוב - חילוק במספרים טבעיים
- יחידה 5: חלק מכמות באחוזים
- יחידה 6: שוב חישוב - חיבור וחיסור

**כיתה ו' - שבילים פלוס 17 - כל 8 היחידות (מלאות)**
- יחידה 1: שברים ומספרים עשרוניים על ישר המספרים
- יחידה 2: שוב חישוב – חיבור וחיסור שברים
- יחידה 3: כפל וחילוק שברים - חלק ב
- יחידה 4: שוב חישוב - משולשים
- יחידה 5: כפל וחילוק מספרים עשרוניים
- יחידה 6: שוב חישוב - זוויות
- יחידה 7: בעיות כוללות
- יחידה 8: שוב חישוב - ממוצע וחציון

**כיתה ו' - שבילים פלוס 18 - כל 8 היחידות (מלאות)**
- יחידה 1: יחס
- יחידה 2: קנה מידה
- יחידה 3: שוב חישוב - מדידת זמן
- יחידה 4: בעיות תנועה והספק
- יחידה 5: מספרים ופעולות
- יחידה 6: שוב חישוב - מרובעים
- יחידה 7: חקר נתונים וניתוח סיכויים
- יחידה 8: שוב חישוב - שטח והיקף

**🎓 יסודות - מיומנויות בסיס לכל הכיתות**
- **לוח הכפל**:
  - 4 מושגי יסוד (מהו כפל, החוק החילופי, כפל ב-0 וב-1, לוח הכפל)
  - 11 טבלאות כפל מלאות (0-10)
  - חידון עם שאלות שנוצרות אוטומטית - שאלות חדשות בכל פעם!

**כל יחידה כוללת:**
- 4 מושגי יסוד עם נוסחאות מתמטיות
- 3 דוגמאות עם פתרונות צעד אחר צעד
- 10 שאלות חידון עם הסברים מפורטים

### 🎯 מערכת חידונים אינטראקטיבית
- **יצירת שאלות אוטומטית** - שאלות חדשות בכל פעם בלוח הכפל
- שאלות רב-ברירה
- משוב מיידי על תשובות
- מערכת ניקוד
- הסברים מפורטים עם פתרון צעד אחר צעד
- אפשרות לנסות שוב
- מעקב אחר התקדמות

### 📊 מעקב התקדמות
- שמירה אוטומטית ב-LocalStorage
- ניקוד מצטבר
- דיוק תשובות
- מחוונים חזותיים על התקדמות ביחידות
- עמוד פרופיל אישי

### 🎨 עיצוב
- תמיכה מלאה ב-RTL (ימין לשמאל)
- פונטים עבריים: Assistant (גוף), Heebo (כותרות)
- אנימציות חלקות
- עיצוב מינימליסטי עם רווחים נדיבים
- קלפים מרחפים עם צללים
- אוברלייז גיאומטריים
- עיצוב רספונסיבי (נייד, טאבלט, דסקטופ)

## 🎯 שימוש באפליקציה

**דרך 1: למידה לפי כיתות**
1. **בחר כיתה** - לחץ על כיתה ו
2. **בחר ספר** - בחר אחד מ-3 הספרים: שבילים פלוס 16, 17 או 18
3. **בחר יחידה** - בחר יחידה מתוך הספר שבחרת
4. **למד**:
   - 📚 **מושגי יסוד** - קרא את המושגים והנוסחאות
   - ✍️ **דוגמאות** - עבור על הפתרונות צעד אחר צעד
   - 🎯 **חידון** - בחן את עצמך וצבור נקודות

**דרך 2: יסודות (מיומנויות בסיס)**
1. **לחץ על "יסודות"** - בעמוד הבית
2. **בחר נושא** - לדוגמה: לוח הכפל
3. **למד**:
   - 📚 **מושגי יסוד** - למד את העקרונות
   - 📊 **טבלאות** - צפה בטבלאות המלאות
   - 🎯 **חידון** - תרגל ובחן את עצמך

**מעקב התקדמות**
- לחץ על "הפרופיל שלי" בכותרת לראות את הניקוד והתקדמות שלך
